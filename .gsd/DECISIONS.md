# DECISIONS.md (Architecture Decision Records)

## ADR-001: Adoption of GSD Framework
- **Date**: 2026-03-31
- **Status**: Accepted
- **Context**: Project required tighter process control to avoid "vibecoding" and ensure premium quality.
- **Decision**: Adopt the Get Shit Done (GSD) context engineering layer.
- **Outcome**: Improved consistency, atomic commits, and spec-driven execution.

## ADR-002: BLoC / Cubit for State Management
- **Date**: 2026-03-31
- **Status**: Proposed
- **Context**: Need for predictable state management in a complex medicine discovery app.
- **Decision**: Use BLoC (Business Logic Component) pattern.
- **Outcome**: Separation of concerns, testability, and clear data flow.
