# PharmaFix: Master Project Plan

## 1. Vision & Goals
**PharmaFix** is a next-generation Flutter application designed to bridge the gap between pharmacies and patients. It provides a seamless experience for medicine discovery, stock tracking, and professional pharmacy services.

## 2. Key Stakeholders
*   **Pharmacists**: Manage stock, verify prescriptions.
*   **Customers**: Search for medicines, locate pharmacies, book services.

## 3. High-Level Core Features
1.  **Medicine Discovery**: Advanced search (Arabic/English), filters (dosage, price).
2.  **Stock Management**: Real-time inventory tracking for pharmacies.
3.  **Prescription Service**: Scan and upload prescriptions.
4.  **Pharmacy Locator**: GPS-based map showing nearby pharmacies.
