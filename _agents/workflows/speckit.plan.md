# /speckit.plan

1. **Review** the Specification from `_planning/specs/`.
2. **Draft** a technical approach in an `implementation_plan` artifact.
3. **Save** a copy to `_planning/plans/PLAN_XXX_{FEATURE}.md`.
4. **Define** the tech stack, data models, and logic flow.
