# /speckit.implement

1. **Pick** the next task from `_planning/tasks/TASK_XXX_{FEATURE}.md`.
2. **Execute** the code changes sequentially.
3. **Verify** each step using `npx -y vitest` or `flutter test` (as project-appropriate).
4. **Mark** tasks as [x] COMPLETED in the task list.
5. **Update** `_planning/MASTER_PLAN.md` progress.
